# Project: Srinidhi BS Personal Website

## Overview
React-based personal website showcasing informational content and PDF utility tools.
Styled with TailwindCSS.

## Architecture
- **Frontend**: React (Create React App)
- **Styling**: TailwindCSS
- **Routing**: React Router
- **PDF Processing**: pdf-lib, pdfjs-dist (client-side)

## Build & Run
- `npm start`: Run dev server
- `npm test`: Run tests
- `npm run build`: Build for production

## Key Features
- Informational pages (Home, Finance, Trading, Travel)
- PDF Tools (Merge, Split, PDF↔JPG conversion, PDF Resizer)
- Dark mode support
- Responsive design
- **Animations**: Framer Motion integration
